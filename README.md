# HOW TO USE 
1.拡張機能(YTMusic-Unity-Sender1.0.0.xpi)をFireFox追加<br/>
2.UnityPackage又は、csファイルをUnityにインポート<br/>
3.https://music.youtube.com で曲を聴く。終わり！閉廷！…以上！皆解散！<br/>
